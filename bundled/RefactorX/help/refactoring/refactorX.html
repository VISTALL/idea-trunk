Refactor-X
The Refactor-X plugin extends IntelliJ IDEA with advanced automated refactorings for XML. The following refactorings are included with Refactor-X
  • Rename Tag
  • Rename Attribute
  • Delete Tag
  • Delete Attribute
  • Unwrap Tag
  • Wrap Tag
  • Wrap Tag Contents
  • Attribute To Tag
  • Tag To Attribute
  • Add Tag
  • Add Subtag
  • Change Attribute Value
  • Convert Contents to Attribute
  • Move Attribute In
  • Move Attribute Out
  • Collapse Tag (implemented as an intention)
  • Expand Tag (implemented as an intention)
  • Split Tag (implemented as an intention)
  • Merge Tags (implemented as an intention)
