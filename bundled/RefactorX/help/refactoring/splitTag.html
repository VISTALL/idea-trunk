Split Tags Intention
The Split Tag intention allows you split a large XML tag into two smaller XML tags. To split tags, simply select the child tag you wish the split to occur on in the body of the tag, and invoke the intention with Alt-Enter. Then select "Split Tags" from the menu, and the tag will be split. Splitting a tag will create two sequential tags, both with the same name and attributes as the original tag. The contents of the first tag will be whatever comes before the selected child tag, with everything else becoming the contents of the second tag.
