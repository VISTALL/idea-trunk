Generate Java Code From Schema using XmlBeans

This topic describes the options available for generating Java Code From Xml Schema using XmlBeans
  • Schema path - file to be used for generation.
  • Output path - name of jar file where generated java code will be placed.
  • Add required client libraries to classpath - generated code depends on additional XmlBeans client libraries. If this option is set then they will be added to the classpath of the module that will contain the generated code.
