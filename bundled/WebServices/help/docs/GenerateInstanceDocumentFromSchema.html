Generate Instance Document from Schema

This topic describes the options available for generating instance document from schema
  • Schema path - path to schema file to be used for xml file generation.
  • Instance document name - name of the instance document to be used for xml file generation.
  • Element name - the local name of the global element to use as the root of the generated document.
  • Enable restriction check - honor restriction particles (if schema uses them).
  • Enable unique element check - honor uniqueness particles (if schema uses them).
