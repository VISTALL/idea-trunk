Generate Schema From Instance Documents

This topic describes the options available for generating Schema From Instance Documents.
  • Instance document path - path for source document for generation.
  • Result schema file name - path to output schema file name for generation.
  • Design type - how to declare elements and complex types (e.g. local elements/types)
  • Detect simple content types - which type to use for leaf text, string or smart (detect from text used)
  • Detect enumeration limit - how many occurences cause schema enumeration to appear (0 - NEVER)
