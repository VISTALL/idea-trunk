Generate Xml Schemas From Class Using JAXB

This topic describes the options available for generating Xml Schema From Class using JAXB.
  • Name of class - class for the generating.
  • Include parameter and return types of the following methods - if selected instructs to use all referenced types for selected methods in order to generate the schema. If not selected only this class and its property types will be created during schema generation.
