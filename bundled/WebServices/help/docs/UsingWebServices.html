Using Web Services

This topic describes the step-by step instructions on how to use Web Services with IntelliJ IDEA.
  • Run Generate Java from WSDL (Tools | WebServices | Generate Java Code From Wsdl) to generate client stubs for typed web service invocation.
  • It is possible to generate web service invocation code by invoking Code | Generate action. The alternative way of such generation is to invoke live template, e.g. for Axis client code one can type itws (invoke typed web service in Axis) in java / jsp and press Tab, live template will insert web service invocation code, like following: 
    
            <WebServiceName>ServiceLocator locator = new <WebServiceName>ServiceLocator();
            <WebServiceName> service = locator.get<WebServiceNameURI>();

            //If authorization is required
            ((<WebServiceNameURI>Soap_BindingStub)service).setUsername("user3");
            ((<WebServiceNameURI>Soap_BindingStub)service).setPassword("pass3");

            // business methods
            System.out.println(service.getIndex(1));
  • It is possible to invoke web service without generating client code, one should activate Enable Web Service Support action (Tools | WebServices | Enable Web Services Support) for needed module with Client setting. After that it is possible to generate web service invocation code by invoking Code | Generate action or, alternatively, type live template name (e.g. iuws) in Editor and press Tab (expand live template), live template will insert web service invocation code.
