Q: When selecting XFire web service platform I got following message Set valid XFire 1.X path in plugin settings

A: WebServices plugin has only Axis 1.4 / JAX-WS RI 2.1 web service platforms bundled so path to any other web services platforms (XFire, JWSDP, Axis2, etc) needs to be configured in plugin settings (Settings | WebServices).
