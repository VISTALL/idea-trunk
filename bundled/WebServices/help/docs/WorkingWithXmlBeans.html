Working with XmlBeans

This topic describes how to work with XmlBeans.
  • Generate java from XmlBeans schemas (Tools | XmlBeans | Generate Java from XmlBeans Schemas).
  • Generate schema from xml document instance (Tools | XmlBeans | Generate Schema From Xml Document Instance).
  • Generate xml document from schema (Tools | XmlBeans | Generate Xml Document from Schema).
  • It is possible to generate marshalling / unmarshalling XmlBeans code by invoking Code | Generate action. The alternative way of such generation is to use live templates mxb (marshal xml bean) and uxb (unmarshal xml bean) to convert XmlBean object to xml and back.
  • Set XmlBeans path in Settings | WebServices
