The Knopflerfish framework integrator provides integration for Knopflerfish.

The Base Directory

The base directory for Knopflerfish framework instances is the installation folder of Knopflerfish. There should be a folder named knopflerfish.org in that folder, which again should contain a folder named osgi. The integrator searches for bundles in a child folder jars and for the matching sources in bundles.
