Set up a new project

You are here:  Osmorc / Set Up / Set up a new project Next | Previous

This guide describes how to set up a new project with Osmorc. Please take a look at Importing an Eclipse Workspace. if you want to import an existing Eclipse workspace into IDEA.

In this section we'll describe how to activate Osmorc for a module. Nothing about the creation of projects and modules which will use Osmorc is Osmorc-specific. So create them just like you would create any other IDEA projects and modules.

To actually activate Osmorc for a module the Osmorc facet needs to be added to the module. Adding the facet can be achieved via two different ways:

Explicit addition of the Osmorc facet
  1. You can manually and explicitly add the Osmorc facet to a module. To do it open File | Settings | Project Settings | Project Settings
  2. Choose modules
  3. Click on the module to which you want to add the facet
  4. Click on the +-icon above the modules tree and choose Osmorc
  5. Now the facet is added and a manifest file is created in the META-INF folder of the module if doesn't exist yet.
Automatic detection of the facet.
  1. Create a folder named META-INF in the root folder of your module.
  2. Create a manifest file MANIFEST.MF inside this folder
  3. At at least the two headers Bundle-ManifestVersion and Bundle-SymbolicName into the manifest file.
  4. IDEA will now recognize the Osmorc facet for the module and will propose to add it. Confirm the addition.

Now you can start working on your OSGI application. Change your manifest files to export and import packages and Osmorc will create corresponding dependencies between the IDEA modules.

You may also want to take a look at the Settings-part of this documentation to learn how to define and use framework instances.
