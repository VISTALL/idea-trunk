Project Settings

The Project Settings tab of Osmorc's settings dialog contains settings specific to the current project..

Currently the only thing you can change here is the framework instance used in the current project. The Framework Instance comboBox contains the names of all framework instances defined for the currently running IntelliJ IDEA. Choose one of them or change to Framework Definitions if the one you need is not yet defined.

  The entries shown in the comboBox are the names of the framework instances followed by the name of the used framework integrator in parentheses. In the case that you are opening a project created with another installation of IntelliJ IDEA that does not yet have the framework instance used by the project, undefined will be shown where normally the name of the framework integrator appears. If that is the case, change to Framework Definitions and define the framework instance.
