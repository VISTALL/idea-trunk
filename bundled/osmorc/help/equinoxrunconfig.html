The Eclipse Equinox run configurations allows running of Eclipse Equinox applications and products.

The first thing to decide in this kind of run configuration is which application or product will be started. If there is an application myApplication defined in the plugin.xml of a bundle with the symbolic name com.mycompany, you would type in com.mycompany.myApplication as the application name. It works the same way for Equinox products.

You can also specify whether the Equinox console shall be activated and whether extra debug output shall be printed on the console. The debug output will - when activated - also contain the log output when starting of an application or product fails, which normally contains details about why it failed.
